Derniers projets

  • FOX CSS

    FOX CSS est un framework CSS poids-plume dont j'ai eu l'idée en voyant la complexité et la taille que certains frameworks connus pouvaient posséder. L'idée était d'avoir une base simple, flexible et modulable, tournée vers le responsive design.

  • FOX Boilerplate

    FOX Boilerplate offre une base de départ pour tout projet web, incluant : création d'un serveur local, compilation et autopréfixage des fichiers Sass, test des fichiers JS, HTML et CSS, livereload, et simplification de la mise en production.
