FOX Boilerplate

A Gulp & Sass Boilerplatelightweight and easy to use - give it a try !

View on GithubDownload as zip

  • Gulp

    FOX Boilerplate is built on top of gulp.js, a powerful JavaScript task manager. Gulp is perfect for automating repetitive tasks.

    Learn more

  • Sass

    Sass is a CSS preprocessor which allows for a really richer CSS syntax. It is also fully integrated within FOX Boilerplate.

    Learn more

  • Autoprefixer

    Autoprefixer is a nice tool which parses CSS and adds vendor prefixes according to the latest rules from the "Can I Use" site.

    Learn more

  • FOX CSS

    FOX CSS is a light CSS framework with smooth resets allowing for a strong flexibility. It also comes with a Sass version.

    Learn more

  • Testing tools

    Your files can be tested with JSHint, CSS Lint and HTML Lint in order to ensure that your code is as clean as possible.

    Learn more

  • Livereload

    This tool (Chrome extension) watches for changes in your browser and will automatically refresh it accordingly.

    Learn more

  • Local server

    Upon running the default task, a tiny local server will be created at localhost:1337 to make your development easier.

    Learn more

  • Notifications

    FOX Boilerplate also comes with an optional notification system (this may require a third party notification software: Growl).

    Learn more
