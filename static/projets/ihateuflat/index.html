CSS3 has brought awesome new features, amongst which gradients and shadows. Nonetheless, the big trend today is FLAT design. Where did all those shiny new effects go ?

I hate U flat ! has been built around the idea that buttons should really stand out – no statistics or market studies here, just a personal opinion. On a side note : I don't really hate flat design; this project was built for fun. Any questions ?

Questions & Support

How does it work ?
It's pretty straightforward : first you need to assign a .btn class to any link, button or submit button you want to customize, and then you can add the specific class you like. The system is quite flexible, just take a look at how the Github button located in the header was built.

What is the browser support ?
The buttons will work on every browser up to old IE's. They will degrade gracefully on older browsers, so you can use them without worrying about them breaking everything.

Any license ?
Yes. This project is under a CC BY 3.0 license. It means that you can do whatever you want with it, as long as you give me proper credits. The icons are coming from Icomoon.

My icons are not showing...
You need to make sure that you have put the Fonts folder at the right place, since it's the one containing the icons fonts (please see the example on Github). Icons also won't show if you're on IE7 or less.
