Friday Links #1

Pour inaugurer la série des Friday Links, je vous propose deux démos de templates de site, un slider uniquement réalisé en CSS, un outil de partage en ligne et un plugin utile pour Sublime Text.

1. Sublime CSS Selector Reveal

On commence par un petit plugin utile pour Sublime Text 2 : CSS Selector Reveal permet, quand on sélectionne un crochet fermant dans notre CSS, d'afficher dans la barre de statut la règle CSS correspondante. Plutôt pratique, surtout si le CSS en question est minifié. Par @LeeMallabone.

Voir le plugin sur Github

2. Codeshare.io

Partagez votre code directement sur votre navigateur. Le code peut être édité par plusieurs personnes en même temps. Développé par @leemunroe.

Voir le site

3. Démo : template de site avec background fixe au scrolling

En ces temps d'effets parallaxe à tout va, voilà une démo sympathique de background fixe au scroll, qui utilise la propriété background-attachment: fixed.Le rendu est vraiment bien, j'essayerai d'intégrer ça dans un futur projet :) Démo par @codrops.

Voir la démo

4. Démo : template de site full screen avec transitions par pages

Une nouvelle démo par les savants fous de chez Codrops. La page initiale est divisée en 4 parties. Quand on clique sur une d'elles, elle s'étend pour occuper toute la taille du navigateur tandis que les 3 autres s'effacent. Le rendu est très sympathique, mais comme souvent avec les transitions ça ne marche que sur navigateurs récents.

Voir la démo

5. Un slider en pur CSS

Gallery CSS est un slider qui n'utilise aucun script. Forcément, cela en limite la compatibilité à IE9 et plus, mais on peut tout à fait imaginer compléter ça avec un fallback sous forme d'image pour IE8 et moins :) Projet par @benschwarz.

Voir le site
