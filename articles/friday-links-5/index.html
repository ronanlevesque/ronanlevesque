Friday Links #5

En me lançant dans mon idée des Friday Links, je m'étais dit que j'essayerais de publier des nouveaux liens toutes les semaines. Bon en fait, c'est plutôt difficile à tenir, surtout en vacances. Du coup l'édition #5 accuse un léger retard... Voici les liens de la semaine :

1. On mobile I want to use

Un site qui permet de connaitre le pourcentage de visiteur sur mobile qui pourraient utiliser votre site en fonction des features que vous souhaitez y intégrer (une large liste de choix est disponible : HTML5 forms, matchMedia, CSS 3D transforms etc.).

Voir le site

2. UI kit

Encore un framework CSS, "léger et modulaire" dixit son introduction. Il est assez complet et intègre en plus un outil de customisation en ligne. Par @yootheme.

Voir le site

3. Démo : toile d'araignée

Une démo utilisant l'élément <canvas> et du JavaScript pour recréer une toile sur laquelle se déplace une araignée; la partie fun étant qu'on peut attraper l'araignée à la souris pour la balader un peu partout. Une démo excellente :)

Voir la démo sur Codepen

4. Tridiv

Un éditeur web pour créer des formes CSS en 3D. L'éditeur est assez impressionnant, et les exemples sur la homepage le sont tout autant :) Ne fonctionne que sous Webkit. Projet par @JulianGarnier

Voir le site

5. ColorHexa

ColorHexa est un site se présentant comme une encyclopédie de la couleur. En rentrant une valeur de couleur (hexa, RGB ou autre) on a accès à un large nombre d'infos sur celle-ci : composition, variation, aperçu par les personnes daltoniennes etc.

Voir le site
