Friday Links #3

Au menu cette semaine : une démo graphique et du pixel art, le tout uniquement en CSS, des outils de prototyping pour animations et interactions, un builder jQuery, et des ressources gratuites pour templates responsive.

1. Démo : une NES en pur CSS

On commence les liens de cette semaine par une démo assez bluffante d'une console NES, rendue uniquement en CSS, pour un rendu très réaliste :) Par @One_div.

Voir la démo sur Codepen

2. Framerjs

Un outil de prototyping pour créer des animations et des interactions, que ce soit sur mobile, tablette ou desktop. Basé sur les @keyframes CSS. Apparemment il s'intègre avec Photoshop, afin d'en exporter directement des éléments. Par @koenbok.

Voir le site

3. PICSSEL ART

Un site qui permet de créer du pixel art en... pur CSS (il se base sur box-shadow pour créer les pixels). Plus gadget qu'autre chose dû à son support limité, mais plutôt sympa comme démo :) Par @chinchang457.

Voir le site

4. jQuery Builder

Ce builder permet de créer une version customisée de jQuery en sélectionnant seulement les modules dont on a besoin (ajax, css, dimensions, effects, offset). Par @JGAui.

Voir le site

5. Ressource : templates responsive gratuits

Pour clore les liens de cette semaine, voici HTML5 Up, un site qui permet de télécharger des templates HTML5 & CSS responsive. Ils sont bien entendus complètement customisables. Projet par @n33co.

Voir le site
